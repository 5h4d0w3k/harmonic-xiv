# XIVHarmonic

A plugin for [XIVLauncher](https://github.com/goatcorp/FFXIVQuickLauncher)/[Dalamud](https://github.com/goatcorp/Dalamud) to dynamically play music tracks depending on various conditions, such as: in-game weather and time of day, combat state, presence of enemies or other players, and various other configurable conditions. 

Currently requires [Orchestrion](https://github.com/perchbirdd/OrchestrionPlugin). Changes are limited to tracks available in-game. A custom music playing mechanism for custom tracks might be added in the future.
